# 🐶🐱 여기서 찾개냥 - 반려동물 찾기 웹사이트  

## 📌 프로젝트 개요  
- **프로젝트명**: 여기서 찾개냥  
- **개발 기간**: 2024.10.28 ~ 2024.11.13  
- **팀 프로젝트**: 부산IT교육센터 (개발 인원: 5명)  
- **목적**:  
  반려동물을 키우는 가구 수가 증가하는 반면, 반려동물 실종 시 이를 효과적으로 찾을 수 있는 서비스는 부족합니다.  
  ‘여기서 찾개냥’은 실종된 반려동물을 찾고, 주인을 잃은 반려동물이 다시 집으로 돌아갈 수 있도록 돕는 웹 서비스입니다.  
  이를 통해 반려동물과 주인의 안전과 행복을 추구하는 **사회적 가치**를 창출하고자 합니다.  

---

## 🛠️ 개발 환경  
- **언어**: Java 17  
- **빌드 도구**: Gradle  
- **프레임워크**: Spring Boot 3.3.5  
- **라이브러리**:  
  - Spring Security  
  - Spring Data JPA  
  - Thymeleaf (Extras & Layout Dialect)  
- **DBMS**: MySQL  
- **개발 도구**: IntelliJ IDEA  

---

## 🔥 주요 기능  

### 1️⃣ 내 반려동물 찾기  
- 사용자가 실종된 반려동물 정보를 등록하여 도움을 요청할 수 있음  
- 반려동물의 특징, 실종 위치, 실종 날짜 등 세부 정보 입력  

### 2️⃣ 반려동물 주인 찾기  
- 사용자가 발견한 반려동물 정보를 등록하여 주인을 찾을 수 있도록 지원  
- 반려동물의 상태, 발견 장소, 특징 등을 입력  

### 3️⃣ 커뮤니티 게시판  
- 반려동물 관련 정보 및 팁 공유  
- 반려동물 실종 예방 및 찾기 경험 공유  

### 4️⃣ 관리자 기능  
- 회원 관리 및 게시글 모니터링  
- 부적절한 게시글 신고 및 삭제 기능
